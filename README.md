# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능목록
* 공격 숫자 선정
  - [x] 1~9 서로 다른 세 자리 랜덤 수 선정
* 수비 숫자 선정
  - [ ] 사용자 입력 수신 CUI
  - [ ] 사용자 입력 검증
* 스트라이크/볼/포볼 판별
  - [ ] 스트라이크 판별
  - [ ] 볼 판별
  - [ ] 포볼 판별
* 판별 결과 출력
  - [ ] 판별 결과 출력 CUI
* 게임 종료/재개 분기
  - [ ] 게임 종료/재개 실행흐름 구성
  - [ ] 게임 재개 여부 선택 CUI
